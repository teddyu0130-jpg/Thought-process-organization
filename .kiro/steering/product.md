# Product Overview

DX施策の検討過程・意思決定の経緯を可視化・構造化し、チーム内で共有可能にするWebアプリ。
テーマ単位でマインドマップ的に思考を可視化し、「なぜこの施策を選んだか」の意思決定理由をたどれるようにする。

**ターゲット**: DX推進チーム（生産技術部門、数名）

## Core Capabilities

1. **テーマ管理** — DX施策テーマの作成・一覧・管理
2. **マインドマップ可視化** — ノード＋エッジによるインタラクティブな思考構造の表現
3. **意思決定メタデータ** — 各ノードに「判断理由」「却下理由」「関連情報」を付与
4. **AI思考整理補助** — 散文的なメモ入力→構造化ノード分解、判断軸の問いかけ、要約生成
5. **ローカル永続化** — プロト段階はlocalStorage/JSONでDB不要

## Target Use Cases

- テーマ（例：トレーサビリティ導入、KPIダッシュボード構築）を起点にツリーを展開し、検討過程を記録する
- 「なぜこの施策を選んだか」「何を検討して何を捨てたか」を後から追跡する
- 途中参加のメンバーが意思決定経緯をキャッチアップする
- 社内提案・デモのためのプロトタイプとして動作検証する

## Value Proposition

個人の頭の中に閉じていたDX施策の思考プロセスを、AIの補助を使ってチーム共有可能な構造化データとして外部化する。
単なるメモツールではなく、「なぜ」を記録することで同じ議論の繰り返しを防ぐ。
